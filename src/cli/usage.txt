   O    O      Carbonyl is a Chromium based browser for the terminal.
    \  /       
O —— Cr —— O   In addition to the following options,
    /  \       Carbonyl also supports most Chromium options.
   O    O      

Usage: carbonyl [options] [url]

Options:
    -f, --fps=<fps>            set the maximum number of frames per second (default: 60)
    -z, --zoom=<zoom>          set the zoom level in percent (default: 100)
    --viewport=<WIDTHxHEIGHT>  override the CSS viewport Chromium lays out against
                               (also via CARBONYL_VIEWPORT env var). When set, the
                               terminal samples a cells * (2, 4) window of the
                               resulting physical raster. Use for reproducible
                               layout across terminal sizes.
    --page-height=<N>          lay the page out N pixels tall for full-page
                               capture (also via CARBONYL_PAGE_HEIGHT env var).
                               --viewport=WxH clips the page to H px; below-the-
                               fold content is never rastered. This overrides
                               only the height, so Chromium rasters the full
                               page and the screenshot capture sees all of it.
                               The interactive terminal still shows the top-left
                               window — use for automation / screenshots.
    --chrome-rows=<N>          stack the URL/nav chrome across N terminal rows
                               (default: 1; also via CARBONYL_CHROME_ROWS env
                               var). Use at large terminal sizes where a
                               single-row chrome smears; try 3 at 360x100.
    --tab-focus                forward Tab and Shift+Tab to Chromium for page
                               focus traversal (default: disabled; alias:
                               --enable-tab-nav; also via CARBONYL_TAB_FOCUS=1).
    --basic-auth=<user:pass>   add Basic Auth userinfo to the first HTTP(S) URL
                               before launching Chromium. The flag is consumed
                               by Carbonyl and is not forwarded to Chromium.
                               Also via CARBONYL_BASIC_AUTH=user:pass.
    --download-dir=<PATH>      set the headless download target directory
                               (also via CARBONYL_DOWNLOAD_DIR env var).
    --file-dialog-path=<PATH>  answer headless file/folder picker prompts with
                               PATH (also via CARBONYL_FILE_DIALOG_PATH env
                               var). Without it, headless pickers cancel.
    --dump-text[=<mode>]       skip the terminal renderer entirely; load the
                               URL, wait for the page, dump page text to
                               stdout, and exit. Modes: innertext (default),
                               accessibility (falls back to innertext until
                               #4 lands), raw-dom. Also via CARBONYL_DUMP_TEXT
                               env var. See docs/runtime-modes.md Mode 4.
    --dump[=png],              skip the terminal renderer entirely; load the
    --screenshot[=png]         URL, wait for compositor frames to stop changing,
                               write the current frame as PNG to stdout, and
                               exit. Also via CARBONYL_DUMP_FRAME=1.
    --idle=<ms>                with --dump-text: wait <ms> after the load
                               event before extracting; with --dump: wait <ms>
                               after the last compositor frame (default: 500).
    --max-wait=<ms>            with dump modes: hard timeout in ms
                               (default: 30000).
    -b, --bitmap               render text as bitmaps
    -d, --debug                enable debug logs
    -h, --help                 display this help message
    -v, --version              output the version number

Keyboard shortcuts:
    Alt+Left / Cmd+Left        go back
    Alt+Right / Cmd+Right      go forward
    Alt+Up / Cmd+Up            toggle color inversion (Alt on Linux/Windows,
                               Cmd on macOS; useful for dark pages in a light
                               terminal and vice versa)

Environment variables:
    CARBONYL_VIEWPORT=WxH      same as --viewport
    CARBONYL_PAGE_HEIGHT=N     same as --page-height
    CARBONYL_CHROME_ROWS=N     same as --chrome-rows
    CARBONYL_TAB_FOCUS=1       same as --tab-focus
    CARBONYL_BASIC_AUTH=u:p    same as --basic-auth; removed from the spawned
                               Chromium child environment
    CARBONYL_DOWNLOAD_DIR=PATH same as --download-dir when the CLI flag is
                               absent
    CARBONYL_FILE_DIALOG_PATH=PATH
                               same as --file-dialog-path when the CLI flag is
                               absent; removed from the spawned Chromium child
                               environment
    CARBONYL_ENV_DEBUG=1       same as --debug
    CARBONYL_ENV_BITMAP=1      same as --bitmap
    CARBONYL_DUMP_TEXT=<mode>  same as --dump-text (empty / "1" implies
                               default mode); CLI flag wins if both set
    CARBONYL_DUMP_FRAME=1      same as --dump=png
    CARBONYL_DUMP_IDLE_MS=<ms> same as --idle
    CARBONYL_DUMP_MAX_WAIT_MS=<ms>
                               same as --max-wait
    CARBONYL_X_MIRROR=1        with --ozone-platform=x11, also blit
                               compositor frames into a visible X window
                               on $DISPLAY so external tools (scrot,
                               ffmpeg, x11vnc) can capture them alongside
                               the terminal render. See docs/runtime-modes.md.

Chromium options:
    Any flag Carbonyl does not recognize is forwarded to Chromium, so you can
    set the proxy, user-agent, language, and most other Chromium switches:
        carbonyl --proxy-server=socks5://127.0.0.1:9050 https://example.com
        carbonyl --user-agent="MyAgent/1.0" https://example.com
        carbonyl --lang=fr-FR https://example.com

Runtime modes (see docs/runtime-modes.md for the full operator guide):
    terminal-only              carbonyl <url>
    x11 + trusted input        carbonyl --ozone-platform=x11 <url>
    x11 + visual capture       CARBONYL_X_MIRROR=1 carbonyl --ozone-platform=x11 <url>
    dump-text                  carbonyl --dump-text <url>
       (text-only extraction)  (see docs/runtime-modes.md Mode 4)
    dump-frame                 carbonyl --dump --viewport=1280x800 <url> > page.png
       (PNG frame capture)     (see docs/runtime-modes.md Mode 5)
